Searching the Inventory

Introduction

Searching for items in the inventory allows for items to be found faster, and thus, makes using Inventory Manager much more efficient.

  • Where Do I Search for Items?
  • Searching for Items

Where Do I Search for Items?

In the items view, there is a text field labeled "Search" at the top of the screen. Fill this text field to filter the table for items matching the input.

Searching for Items

In the search bar at the top of the items view, the user can enter in a value matching an item's name or serial number to filter the table for that specific item. The search bar looks for both names and serial numbers, so it is advised that if the user is only looking for one particular item that they enter the entire serial number of that item. Whenever the button to the right of the search bar is clicked, or if the search bar is emptied, the table will return back to showing all of the items in the inventory.
