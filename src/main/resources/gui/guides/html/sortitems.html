Sorting the Inventory

Introduction

Sorting the items in the inventory just involves clicking one of the three buttons under the table, which will cause the items to be sorted in both the table and the inventory itself.

Where Do I Sort Items?

Under the table in the items view, there are three buttons that are used to sort the inventory. These buttons sort the inventory by name, value, and serial number respectively, and when clicked a second time, will sort the inventory in the opposite order.

Sorting Items

The methods that a user can use to sort the inventory will be explained in greater detail in the sections below. If any of these methods are used, then the items will be sorted when the inventory is exported, and will appear in the same order when imported as well.

Sorting by Name

Clicking the "Sort By Name" button will result in the inventory being sorted in ascending order by name, or in alphabetical order. Clicking the button again will cause the inventory to be sorted in descending order by name.

Sorting by Value

Clicking the "Sort By Value" button will result in the inventory being sorted in ascending order by value, or in numerical order. Clicking the button again will cause the inventory to be sorted in descending order by value.

Sorting by Serial Number

Clicking the "Sort By Serial Number" button will result in the inventory being sorted in ascending order by serial number, or in alphabetical order. Clicking the button again will cause the inventory to be sorted in descending order by serial number.
