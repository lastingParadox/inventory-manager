Adding Items

Introduction

In the Inventory Manager application, a user can add items through a matter of filling out a few fields and clicking a button.

Where Do I Add an Item?

In the items view, a user can find the ability to add items underneath the sorting buttons beneath the table. In the section prefaced "Create New Item", there are three text fields, representing the item's name, value, and serial number, and a button labeled, "Add Item".

Name

An item's name is used to denote the item, and is a mandatory field that must be filled to add the item to the inventory. All symbols are permitted within an item's name, however the item's name must be between 2 and 256 characters. If the user attempts to add an item with less than 2 characters, the application will produce a popup upon trying to add the item to the inventory, and the item will not be added.

Note that upon trying to write an item's name past 256 characters will result in the field halting any text entry past 256 characters. The character limit counter beneath the field can be used to determine how many characters have been filled out in the name field.

Value

An item's value is a mandatory field that represents the price, or the value, or the item in United States dollars. Only numerical values are permitted in an item's value field, and attempting to input a non-numerical value will result in an error popup appearing whenever the "Add Item" button is clicked.

In addition, the value field of an item must be greater than or equal to 0. The value is purely numerical--entering the item's value with a dollar sign ($) will result in an error.

Serial Number

An item's serial number is also a mandatory field used as a unique moniker for the item. While multiple items may share the same name, serial numbers cannot be shared across items. The item's serial number must be entered in the format "A-XXX-XXX-XXX", where "A" represents a letter and "X" represents a letter or a digit. Attempting to enter a value in the serial field that does not match this format will result in an error popup appearing when clicking the "Add Item" button.

Attempting to enter a serial number that is identical to another item's serial number will also result in an error popup appearing. As previously mentioned, items' serial numbers are unique across an inventory.

Adding the Item

After all three item fields are filled, clicking the "Add Item" field will result in either a popup appearing denoting any validation errors in the field inputs or in the item being added to the table above. If a popup appears, the user must fix the text fields associated with the errors for the item to be shown in the table. Upon a successful addition, the text fields will be cleared, and the newly created item will be added to the inventory for later modification.
