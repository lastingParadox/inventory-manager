Importing and Exporting the Inventory

Introduction

In order to be able to use an inventory after closing the program, the user must export their inventory, and import it when using Inventory Manager later. Luckily, exporting and importing an inventory is a simple task.

Where Do I Export and Import?

In the side menu, there is a section titled "Import and Export", which contains four buttons: "Import Items", "Export as Text (TSV)", "Export As HTML", and "Export as JSON". In addition, the menu at the top contains the "File" menu option, which can be used to import or export. Each of these buttons are used in either loading the inventory to the table or saving the inventory to an external file in a variety of formats.

Importing an Inventory

Providing the user has previously exported an inventory, clicking the "Import" button will result in a window appearing to select a file. Any inventory file can be chosen, or any text, html, or json file. However, in order for the file to be processed correctly in the table, the file chosen should have been previously created by the Inventory Manager program. With a successful import, the application will show a confirmation message, and the items in the file that was imported will populate the table for manipulation.

Exporting an Inventory

After the user has an inventory created, they can export their inventory as either a text file, a HTML file, or a JSON file. For each of these methods, the user will be prompted with a window to choose the location of the inventory file and its name. After clicking "Save", the file will be created, and a confirmation window will appear, mentioning that the file was successfully exported.

Export as Text

If the user wishes to save their inventory as a text file, the program will write the data in the inventory in a TSV format. This text file can be used in other applications that can read tab-separated values, such as Microsoft Excel.

Export as HTML

If the user wishes to save their inventory as a HTML file, the program will write the data in the inventory in a table in an HTML format. The table created will look similar to the table below, and can be used in other HTML-related applications, particularly websites.

Export as JSON

If the user wishes to save their inventory as a JSON file, the program will write the inventory in a JSON format. The JSON file can be used in applications that read in JSON files, particularly Javascript.
